🎓 INF.02 - Baza Wiedzy

Kompletne materiały do egzaminu zawodowego. Teoria, laboratoria i praktyka w jednym miejscu.

🟢

Filar 1: Sieci

Packet Tracer, adresacja IP, VLANy, routing, DHCP, DNS. Od podstaw L2/L3 do zaawansowanych topologii.

📚 Teoria + 3 Labs ⏱️ ~8h
🪟

Filar 2: Windows Server

Active Directory, GPO, NTFS, udziały, DNS, DHCP, IIS. Wszystko o zarządzaniu siecią domenową.

📚 Teoria + 5 Labs ⏱️ ~12h
🐧

Filar 3: Linux SysOps

Terminal, uprawnienia, dowiązania, diagnostyka sprzętu. Niezbędne komendy dla administratora.

📚 Teoria + 3 Labs ⏱️ ~6h
🔌

Filar 4: Warstwa Fizyczna

Okablowanie strukturalne, RACK, diagnostyka SMART, GPU-Z. Praktyczne umiejętności techniczne.

📚 Teoria + 2 Labs ⏱️ ~4h
📊

Filar 5: Excel & Logika

Formuły, funkcje JEŻELI, formatowanie warunkowe, kosztorysy. Praktyczne arkusze dla biznesu.

📚 Teoria + Ćwiczenia ⏱️ ~3h
