Proporcje tematów na egzaminie

🟢 Sieci Komputerowe ~35%

  • Adresacja IP (IPv4, IPv6)
  • Maski podsieci i subnetting
  • VLANy i segmentacja sieci
  • Routing (statyczny i dynamiczny)
  • Protokoły: TCP, UDP, ICMP, ARP
  • DHCP, DNS, NAT
  • Cisco Packet Tracer

🪟 Windows Server ~30%

  • Active Directory Domain Services
  • Zasady grupy (GPO)
  • Uprawnienia NTFS
  • Udostępnianie plików (SMB/CIFS)
  • Profile użytkowników (lokalne, mobilne)
  • Usługi: DNS, DHCP, IIS, RRAS
  • Komendy CMD: net user, net group, ipconfig

🐧 Linux ~15%

  • Nawigacja w terminalu (cd, ls, pwd)
  • Operacje na plikach (cp, mv, rm, touch)
  • Uprawnienia (chmod, chown)
  • Dowiązania (ln, ln -s)
  • Archiwizacja (tar, gzip)
  • Diagnostyka: lshw, lsblk, df, top
  • Przekierowania wyjścia (>, >>)

🔌 Warstwa Fizyczna ~10%

  • Okablowanie strukturalne (T568A/B)
  • Moduły Keystone Jack
  • Szafy serwerowe (RACK 19")
  • Diagnostyka dysków (SMART)
  • Monitorowanie GPU (GPU-Z)
  • Kategorie kabli (Cat5e, Cat6, Cat6a)

📊 Excel & Logika ~10%

  • Formuły i funkcje podstawowe
  • Funkcja JEŻELI i zagnieżdżenia
  • Formatowanie warunkowe
  • Odwołania względne i bezwzględne ($)
  • Kosztorysy i obliczenia biznesowe
  • Wykresy i wizualizacje
💡
Strategia nauki: Skoncentruj się na Sieciach i Windows Server (razem ~65% egzaminu). Linux i warstwa fizyczna to mniejsze części, ale często pojawiają się w zadaniach praktycznych. Excel jest zazwyczaj najłatwiejszą częścią - nie zaniedbuj jej, to łatwe punkty!
