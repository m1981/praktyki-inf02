Routing, Niezawodność i Drukarki

Część 1: Serwer jako Router (RRAS i NAT)

  • Krok 1: Dwie karty sieciowe

    Wyłącz maszynę z Windows Serverem. W ustawieniach VirtualBox dodaj drugą kartę sieciową i ustaw ją na NAT (to będzie nasz WAN). Pierwsza karta zostaje na Internal Network (Sieć wewnętrzna) (to nasz LAN).

  • Krok 2: Instalacja roli RRAS
    Server Manager (Menedżer serwera) ➔ Add roles and features (Dodaj role i funkcje) ➔ Remote Access (Dostęp zdalny) ➔ Routing
  • Krok 3: Konfiguracja Translacji Adresów (NAT)
    Routing and Remote Access (Routing i dostęp zdalny)

    Kliknij prawym przyciskiem na nazwę serwera i wybierz "Configure and Enable Routing and Remote Access" (Konfiguruj i włącz routing...). Wybierz Network Address Translation (NAT) (Translacja adresów sieciowych (NAT)).

    • 🚨 Najczęstszy błąd uczniów (RRAS): W kreatorze NAT system zapyta, który interfejs jest podłączony do Internetu (Publiczny). Jeśli wskażesz kartę LAN zamiast WAN, odetniesz całą sieć od świata! Zawsze nazywaj karty sieciowe w systemie (np. "Karta_LAN", "Karta_WAN") przed konfiguracją RRAS.

Część 2: Niezawodność (NIC Teaming)

  • Krok 1: Zespół kart sieciowych
    Server Manager (Menedżer serwera) ➔ Local Server (Serwer lokalny) ➔ NIC Teaming (Zespół kart interfejsu sieciowego)

    Zaznacz dwie wewnętrzne karty sieciowe, kliknij prawym -> Add to New Team (Dodaj do nowego zespołu). Nazwij go LAN_Zespol. Teaming mode (Tryb): Switch Independent (Niezależny od przełącznika).

    • 💡 Złota Rada Trenera: Po utworzeniu zespołu (Team), Twoje dwie fizyczne karty tracą swoje adresy IP! Musisz wejść w ustawienia sieciowe (ncpa.cpl) i nadać adres IP nowej, wirtualnej karcie o nazwie LAN_Zespol.

Część 3: Szkolna Drukarka (Print Server)

  • Krok 1: Instalacja portu TCP/IP
    Control Panel (Panel Sterowania) ➔ Devices and Printers (Urządzenia i drukarki) ➔ Add a printer (Dodaj drukarkę)

    Wybierz "Add a printer using a TCP/IP address or hostname" (Dodaj drukarkę używając adresu TCP/IP). Wpisz adres, np. 192.168.1.200. Zainstaluj sterownik Generic / Text Only.

  • Krok 2: Udostępnianie i restrykcje

    Wejdź we właściwości nowej drukarki. W zakładce Advanced (Zaawansowane) zmień opcję na Available from (Dostępna od) 08:00 to 15:00.

📸 Weryfikacja Egzaminacyjna (Zrzuty ekranu)

W przypadku zaawansowanych usług, egzaminator wymaga twardych dowodów:

  1. RRAS: Zrób zrzut ekranu ze stacji roboczej (Windows 10), na którym widać udany ping 8.8.8.8 (dowód, że NAT działa).
  2. NIC Teaming: Zrób zrzut okna Menedżera Serwera, gdzie widać Twój zespół kart ze statusem OK na zielono.
  3. Drukarka: Zrób zrzut zakładki "Zaawansowane" we właściwościach drukarki, pokazujący ustawione godziny dostępności.
