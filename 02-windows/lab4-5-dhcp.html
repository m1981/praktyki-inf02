Automatyzacja Adresacji IP

Lab 4.5: Automatyzacja IP (Serwer DHCP)

Czas: 45 min
Cel: Uruchomienie serwera DHCP na Windows Serverze, utworzenie puli adresów oraz rezerwacji dla konkretnego komputera.
Jak działa DHCP (Proces DORA)?

DHCP (Dynamic Host Configuration Protocol) to usługa, która automatycznie rozdaje adresy IP komputerom w sieci. Proces ten nazywa się DORA:

  • D (Discover): Komputer krzyczy: "Halo! Potrzebuję adresu IP!"
  • O (Offer): Serwer odpowiada: "Mam wolny adres 192.168.1.50."
  • R (Request): Komputer mówi: "Biorę go! Zarezerwuj go dla mnie."
  • A (Acknowledge): Serwer potwierdza: "Zapisane. Adres jest twój."
  • Krok 1: Instalacja roli DHCP
    Server Manager (Menedżer serwera) ➔ Add roles and features (Dodaj role i funkcje) ➔ DHCP Server (Serwer DHCP)

    Po instalacji kliknij żółty trójkąt i wybierz "Complete DHCP configuration" (Zakończ konfigurację usługi DHCP) (autoryzacja w domenie).

    • 🚨 Najczęstszy błąd uczniów: Brak autoryzacji serwera DHCP w Active Directory. Jeśli pominiesz ten krok (żółty trójkąt), serwer DHCP w środowisku domenowym nie rozda ani jednego adresu IP!
  • Krok 2: Tworzenie Zakresu (Puli adresów)
    DHCP Manager (Menedżer DHCP) ➔ IPv4 ➔ New Scope (Nowy zakres)

    Rozwiń nazwę serwera -> IPv4. Kliknij prawym na IPv4 i wybierz New Scope...

    • Name (Nazwa): Siec_Szkolna
    • Start IP address (Początkowy adres IP): 192.168.1.100
    • End IP address (Końcowy adres IP): 192.168.1.150
    • Subnet mask (Maska podsieci): 255.255.255.0
  • Krok 3: Opcje Zakresu (Brama i DNS)

    W kreatorze zostaniesz zapytany o opcje DHCP. To bardzo ważne!

    • Router (Default Gateway / Brama domyślna): Wpisz adres IP Twojego routera.
    • DNS Server (Serwer DNS): Wpisz adres IP Twojego Windows Servera.
  • Krok 4: Rezerwacja adresu IP

    Rozwiń swój nowy zakres. Kliknij prawym na Reservations (Zastrzeżenia) -> New Reservation (Nowe zastrzeżenie)...

    Reservation name (Nazwa): Komputer_Dyrektora. IP address: 192.168.1.110. MAC address: wpisz dowolny adres MAC (np. 00-11-22-33-44-55).

    Zobacz oryginalne zadanie z egzaminu

    Egzamin: Czerwiec 2021 (INF.02-01-21.06-SG)

    Arkusz: Pobierz PDF

    Zadanie: 2. Skonfiguruj ruter według następujących zaleceń: (...)

    Treść zadania:
    ‒ serwer DHCP włączony, zakres adresów 172.20.0.100 ÷ 172.20.0.149
    ‒ adres IP 172.20.0.110 zarezerwowany dla karty WiFi stacji roboczej

    Uwaga Trenera: Na egzaminie INF.02 DHCP najczęściej konfiguruje się na Routerze, ale zasada działania (Pula, Rezerwacja po MAC) jest identyczna jak na Windows Serverze!

    • 💡 Złota Rada Trenera (Wymuszanie DHCP): Jeśli stacja robocza nie chce pobrać nowego adresu, otwórz na niej CMD i wpisz kolejno:
    ipconfig /release (oddaje stary adres)
    ipconfig /renew (prosi o nowy adres)

📸 Weryfikacja Egzaminacyjna (Zrzuty ekranu)

Egzaminator musi widzieć, że stacja robocza faktycznie pobrała adres z Twojego serwera:

  1. Na stacji roboczej (Windows 10) otwórz CMD i wpisz ipconfig /all.
  2. Zrób zrzut ekranu tak, aby było widać linijki: IPv4 Address oraz DHCP Server (musi tam być IP Twojego serwera!).
  3. Wklej zrzut do pliku tekstowego na pendrive.
