Automatyzacja Adresacji IP

Lab 4.5: Automatyzacja IP (Serwer DHCP)

Czas: 45 min
Cel: Uruchomienie serwera DHCP na Windows Serverze, utworzenie puli adresów oraz rezerwacji dla konkretnego komputera.
Jak działa DHCP (Proces DORA)?

DHCP (Dynamic Host Configuration Protocol) to usługa, która automatycznie rozdaje adresy IP komputerom w sieci. Proces ten nazywa się DORA:

  • D (Discover): Komputer krzyczy: "Halo! Potrzebuję adresu IP!"
  • O (Offer): Serwer odpowiada: "Mam wolny adres 192.168.1.50."
  • R (Request): Komputer mówi: "Biorę go! Zarezerwuj go dla mnie."
  • A (Acknowledge): Serwer potwierdza: "Zapisane. Adres jest twój."
  • Krok 1: Instalacja roli DHCP
    Menedżer serwera ➔ Dodaj role i funkcje ➔ Serwer DHCP

    Po instalacji kliknij żółty trójkąt i wybierz "Zakończ konfigurację usługi DHCP" (autoryzacja w domenie).

  • Krok 2: Tworzenie Zakresu (Puli adresów)
    Narzędzia ➔ DHCP (dhcpmgmt.msc)

    Rozwiń nazwę serwera -> IPv4. Kliknij prawym na IPv4 i wybierz Nowy zakres...

    • Nazwa: Siec_Szkolna
    • Początkowy adres IP: 192.168.1.100
    • Końcowy adres IP: 192.168.1.150
    • Maska podsieci: 255.255.255.0
  • Krok 3: Opcje Zakresu (Brama i DNS)

    W kreatorze zostaniesz zapytany o opcje DHCP. To bardzo ważne!

    • Router (Brama domyślna): Wpisz adres IP Twojego routera.
    • Serwer DNS: Wpisz adres IP Twojego Windows Servera.
  • Krok 4: Rezerwacja adresu IP

    Rozwiń swój nowy zakres. Kliknij prawym na Zastrzeżenia (Reservations) -> Nowe zastrzeżenie...

    Nazwa: Komputer_Dyrektora. Adres IP: 192.168.1.110. Adres MAC: wpisz dowolny adres MAC (np. 00-11-22-33-44-55).

🔄
Sprawdź się: Wejdź na stację roboczą Windows 10. Zmień ustawienia karty sieciowej na "Uzyskaj adres IP automatycznie". Otwórz CMD i wpisz ipconfig /all. Powinieneś zobaczyć adres z puli .100-.150!
